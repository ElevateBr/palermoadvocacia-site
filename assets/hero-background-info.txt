# Imagem de Fundo da Seção Hero

## Arquivo: images/hero-lawyer.png

### Especificações Técnicas:
- **Tamanho recomendado**: 1920x1080px (Full HD)
- **Formato**: PNG (recomendado para melhor qualidade)
- **Tamanho do arquivo**: Máximo 500KB
- **Qualidade**: Alta resolução, mas otimizada para web

### Características da Imagem:
- **Tema**: Advogado ou escritório de advocacia
- **Estilo**: Profissional e moderno
- **Cores**: Tons neutros ou que combinem com preto (#000000) e marrom (#8B6154)
- **Composição**: 
  - Advogado em traje formal
  - Ou escritório moderno com mesa e livros
  - Ou prédio de advocacia
  - Evitar imagens muito coloridas ou distrativas

### Como Funciona:
- A imagem será exibida como fundo da seção hero
- Terá um overlay escuro (fade) para garantir legibilidade do texto
- O texto ficará em branco sobre o fundo escuro
- A imagem terá efeito parallax (fixed attachment)

### Exemplo de Imagem:
- Advogado de costas olhando pela janela
- Mesa de escritório com documentos
- Fachada de prédio corporativo
- Biblioteca jurídica

### Posicionamento:
- A imagem será centralizada
- Terá `background-size: cover` para preencher toda a tela
- Em dispositivos móveis, o parallax será desabilitado

### Overlay Aplicado:
```css
background: linear-gradient(rgba(0, 0, 0, 0.6), rgba(0, 0, 0, 0.6)), url('images/hero-lawyer.png');
```

Isso cria um fade preto sobre a imagem para garantir que o texto branco seja legível.
